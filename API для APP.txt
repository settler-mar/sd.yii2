Для авторизации как и писал нужно отправить GET запос на https://api.secretdiscounter.com/user/login
В ответ будет строка вида ad8f0c6444877f54b479ffb3b20be285

Следующий POST запрос должен идти на этот же адрес. Передаются 3 поля
- user
- password
- key

Поле key это md5 сумма от объединенной строки
user, appKey, password,get_key
appKey - будет постоянный
get_key - ключ из первого запроса

Если в поле KEY не пройдет валидация то будет 404-я ошибка

Если ключ совпал будет ответ сервера 200 с результатом авторизации
0 - не авторизировало
1 - авторизировало
2 - пользователь удален
3 - ошибка авторизации
